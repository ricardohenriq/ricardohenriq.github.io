About me

Sou bacharelando em Engenharia de Software pela Universidade Federal de Góais (UFG), durante os 3 primeiros periodos da faculdade trabalhei com a linguagem C e posteriormente mais 4 periodos utilizando Java na construção das aplicações utilizadas como trabalho academico, pude trabalhar com Engenharia de Requisitos e com processos gerencias do desenvolvimento usando metodologias ageis como scrum.

Por conta própria estudei sobre desenvolvimento back-end e front-end de sistemas web usando frameworks que visam a criação de uma interface de usuário responsiva, e integração com uma base de dados.

Trabalhei com ferramentas de gestão, projeto e desenvolvimento, juntamente com outras ferramentas de próprositos mais especificos pertinentes quase exclusivamente a implementação de cada aplicação construida.
