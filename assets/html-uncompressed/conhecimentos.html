Back-end

Trabalhei com PHP no desenvolvimento (ainda em andamento) de um um Parser de sites (este parser tem o objetivo de identificar um conjunto similar de imagens e baseado nestas imagens extrair informações relevantes predefinidas respeitando condições de desempenho, após encontradas as informações relevantes são organizadas e salvas automaticamente) e no sistema web responsável por recuperar e repassar as informações aos usuários mediante consulta com diversos filtros.

Front-end

Trabalhei com XHTML/CSS/Javascript para desenvolver o Front-end do sistema web (responsável pelo interfaceamento do usuário e da base de dados que contem os dados provenientes do funcionamento do Parser) juntamente com vários frameworks que forneciam um desenvolvimento mais rápido e métodos mais arrojados.

Desktop

Durante a graduação trabalhei por diversos períodos com o projeto em (UML) e desenvolvimento em Java (não simultaneamente) e desenvolvi vários programas em C e realizei atividades de build de dll e libs de bibliotecas de suporte a OpenCV.

Front-end: Frameworks & Plugins

Trabalhei com vários frameworks e plugins afim prover uma interface rica e responsiva para o front-end que expõe o conteúdo obtido por meio do Parser.

Back-end: Frameworks & Plugins

Utilizo varias bibliotecas externas ao PHP para fornecer funcionalidades como o envio de emails usando os servidores SMTP do google e fornecer uma abstração orientada a objeto para navegar no DOM, também utilizo framework de teste unitário de código PHP.

Banco de dados

Trabalho com banco de dados relacional Mysql desde a graduação onde tive um período inteiro de estudos e trabalhos de uso exclusivo de banco de dados além de utilizá-lo simultaneamente com outras tecnologias (Java), fora da graduação utilizo integrado com PHP para armazenamento e busca dos dados obtidos e organizados pelo Parser (também desenvolvido por mim).

Conhecimento em gerência
