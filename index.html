Sobre mim

Sou bacharelando em Engenharia de Software pela Universidade Federal de Goiás (UFG), durante os 3 primeiros períodos da faculdade trabalhei com a linguagem C e posteriormente mais 4 períodos utilizando Java na construção das aplicações utilizadas como trabalho acadêmico, pude trabalhar com Engenharia de Requisitos e com processos gerencias do desenvolvimento usando metodologias ágeis como Scrum.

Por conta própria estudei sobre desenvolvimento Back-end e Front-end de sistemas web usando frameworks que visam a criação de uma interface de usuário responsiva, e integração com uma base de dados.

Trabalhei com ferramentas de gestão, projeto e desenvolvimento, juntamente com outras ferramentas de propósitos mais específicos pertinentes quase exclusivamente a implementação de cada aplicação construída.
